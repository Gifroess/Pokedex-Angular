## Como rodar

```bash
npm install
npm start
```

Depois, acesse:

```bash
http://localhost:4200/
```

## Build de produção

```bash
npm run build
```

## Testes

```bash
npm test -- --watch=false
```

## Observação

O projeto usa dados da PokéAPI. Para carregar os Pokémon no navegador, é necessário estar conectado à internet.
